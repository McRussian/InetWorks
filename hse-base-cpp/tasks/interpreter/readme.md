# Интерпретатор игрушечного языка программирования

Напишите интерпретатор [некоторого эзотерического языка программирования](https://en.wikipedia.org/wiki/Brainfuck). Несмотря на минимализм (всего 8 команд), данный язык является Тьюринг-полным.

## Модель вычислений

Программа оперирует **лентой** из ячеек памяти и **указателем** на текущую ячейку.
Каждая ячейка хранит один байт (значение от 0 до 255). Изначально все ячейки равны нулю, указатель указывает на первую ячейку.

## Команды

| Символ | Действие |
|--------|----------|
| `>` | Сдвинуть указатель вправо |
| `<` | Сдвинуть указатель влево |
| `+` | Увеличить значение в текущей ячейке на 1 |
| `-` | Уменьшить значение в текущей ячейке на 1 |
| `.` | Вывести значение текущей ячейки как символ ASCII |
| `,` | Прочитать один символ из входного потока и записать в текущую ячейку |
| `[` | Если значение текущей ячейки равно 0, перейти к команде **после** соответствующей `]` |
| `]` | Если значение текущей ячейки **не** равно 0, перейти к команде **после** соответствующей `[` |

Все остальные символы в программе игнорируются (можно использовать как комментарии).

## Детали реализации

- Размер ленты задаётся в конструкторе.
- Тип ячейки: `uint8_t`. Переполнение определено: `255 + 1 = 0`, `0 - 1 = 255`.
- Если указатель выходит за границы ленты, поведение не определено (в тестах такого не будет).
- Если входной поток закончился при выполнении `,`, в ячейку записывается 0.
- Скобки `[` и `]` всегда сбалансированы (в тестах некорректных программ не будет).

## Что нужно реализовать

Класс `Interpreter`. Конструктор принимает размер ленты. Метод `Run` принимает код программы и входные данные, а возвращает строку — всё, что программа вывела через `.`.

Один объект `Interpreter` можно использовать для нескольких запусков. Состояние ленты **сохраняется** между вызовами `Run` — указатель при этом сбрасывается на начало ленты.

## Примеры

### Hello World

```
++++++++[>++++[>++>+++>+++>+<<<<-]>+>+>->>+[<]<-]>>.>---.+++++++..+++.>>.<-.<.+++.------.--------.>>+.>++.
```

Выводит: `Hello World!\n`

### cat (копирование входа на выход)

```
,[.,]
```

### Сложение двух цифр

```
,                          Читаем первую цифру в cell[0]
>,                         Читаем вторую цифру в cell[1]
[<+>-]                     Прибавляем cell[1] к cell[0]
<>++++++[<-------->-]<     Вычитаем '0' (48)
.                          Выводим результат
```

На входе `"35"` выведет `"8"` (коды символов: `'3' + '5' - '0' = 51 + 53 - 48 = 56 = '8'`).
