# Game of Life

[Game of Life](https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life) — клеточный автомат, придуманный математиком Джоном Конвеем в 1970 году. Несмотря на простые правила, в нём возникает удивительно сложное поведение: самоорганизация, движущиеся структуры и даже вычислительная полнота по Тьюрингу.

Реализуйте эффективный движок этой игры.

## Правила

Вселенная Game of Life — двумерная сетка клеток. Каждая клетка может быть **живой** (1) или **мёртвой** (0). На каждом шаге состояние всех клеток обновляется одновременно по следующим правилам:

- **Рождение**: мёртвая клетка с **ровно 3** живыми соседями оживает.
- **Выживание**: живая клетка с **2 или 3** живыми соседями продолжает жить.
- **Смерть**: во всех остальных случаях клетка умирает (или остаётся мёртвой).

Соседи — 8 клеток вокруг: сверху, снизу, слева, справа и по диагоналям.

## Тор

В нашем случае мы реализуем упрощенную версию правил: поле замкнуто в **тор**. Левый край соединён с правым, верхний — с нижним.

```
Сосед слева от (r, 0)      → (r, cols - 1)
Сосед сверху от (0, c)     → (rows - 1, c)
Сосед по диагонали от (0, 0) → (rows - 1, cols - 1)
```

## Интерфейс

```cpp
using Grid = std::vector<std::vector<uint8_t>>;  // 0 = мёртвая, 1 = живая

class GameOfLife {
public:
    explicit GameOfLife(const Grid& initial_state);

    void Simulate(uint64_t steps);

    size_t Rows() const;
    size_t Cols() const;
    bool IsAlive(size_t row, size_t col) const;
    Grid GetState() const;
};
```

Для хранения поля используем `std::vector<uint8_t>` вместо `std::vector<bool>`, так как последний нетривиальным образом упаковывает значения, что может вызывать проблемы.
- `Simulate(1)` — один шаг симуляции.
- `Simulate(n)` — `n` последовательных шагов.
- `GetState()` — текущее состояние поля.

## Визуализатор

В задаче есть готовый визуализатор, который показывает симуляцию в терминале. Чтобы им воспользоваться, соберите таргет `game_of_life_tui` и запустите в терминале (лучше уменьшить размер шрифта):

```bash
cmake --build build --target game_of_life_tui
./build/game_of_life_tui glider
./build/game_of_life_tui pulsar 200    # 200 мс между кадрами
```

Управление:
- `Пробел` — пауза
- `+` / `-` — скорость
- `Q` — выход

Доступные паттерны:

| Паттерн | Описание |
|---------|----------|
| `block`, `beehive` | Стабильные фигуры |
| `blinker`, `toad` | Осцилляторы (период 2) |
| `pulsar` | Осциллятор (период 3) |
| `pentadecathlon` | Осциллятор (период 15) |
| `glider`, `lwss` | Космические корабли |
| `gun` | Пушка Госпера (генерирует глайдеры) |
| `rpentomino` | Хаотичный долгоживущий паттерн |
| `random` | Случайное поле 32x32 |
