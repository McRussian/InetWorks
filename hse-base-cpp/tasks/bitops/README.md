# Битовые операции

В этой задаче вам предстоит реализовать различные битовые операции над 64-битными беззнаковыми целыми числами. Функции описаны в файле [bitops.h](./bitops.h), вам стоит почитать заголовочный файл и тесты, после чего реализовать свои версии функций в [bitops.cpp](./bitops.cpp).

## На что обратить внимание
- Все входные значения помещаются в `uint64_t`
- Для функций сдвига (`shift`) значения могут быть больше 64 — используйте операцию по модулю
- Обратите внимание на особые случаи (например, `value = 0`)
- По возможности, не используйте циклы. Где циклы необходимы, необходимо гарантировать, что цикл для любого числа завершится за константное заранее определенное число операций.
